## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS
- **Icons**: `lucide-react`
- **Markdown parsing**: `react-markdown`, `remark-math`, `rehype-katex`

## Getting Started

First, ensure you have Node.js installed, then install the dependencies (if you haven't already):

```bash
npm install
```

Start the Next.js development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
